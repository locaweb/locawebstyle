Tooltips ou popover

Para mostrar balões de informação.

As opções devem ser passadas via atributo data, ex: data-animation="false", data-trigger="click".

<a data-content="Aqui vai um conteúdo maravilhoso para o popover... aceita &lt;b&gt;html&lt;/b&gt;" rel="popover" data-original-title="Exemplo de Popover"></a>
Name type default description
animation boolean true Aplica uma transição de fade
placement string|function 'right' Onde o popover ficará posicionado - top | bottom | left | right
selector string false Se um seletor é definido, os objetos do popover será delegado para alvos específicos
trigger string 'hover' Como o popover é ativado - hover | focus | manual
title string | function '' Título padrão do popover se o atributo title não existir
content string | function '' Conteúdo padrão se o data-content não existir
delay number | object 0

Delay para mostrar e esconder o popover (ms) - não aplicável para ativação manual.

Se um número é fornecido, o delay é aplicado no hide e show.

Estrutura do objeto é: delay: { show: 500, hide: 100 }

Classe .infoBubble

O .infoBubble faz a caixinha azul com o ponto de interrogação aparecer.

Tooltip dentro de collapse

Se você precisou colocar um tooltip dentro de um collapse, pode ser que o seu tooltip fique cortado, se estiver com este problema, use a classe .with-tooltip no elemento de collapse.

Este é um fix temporário e você deve usar com cuidado. O problema já foi corrigido nas últimas versões do Bootstrap e na próxima versão do Locastyle, a 2.0.
