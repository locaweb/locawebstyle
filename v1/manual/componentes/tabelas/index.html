Tabelas

Exibindo dados tabulares.

Perceba que para deixar os títulos de colunas com marcação de ordenação, basta utilizar as classes dataDescending ou dataAscending.

Tabela no padrão do bootstrap utilizamos a classe .table. Esse tipo de tabela não possue bordas laterais.

Classes que manipulam o estilo da tabela padrão.

Para alterar o estilo da tabela padrão com a classe .table, basta adicionar as classes abaixo:

table-bordered: adiciona borda

table-striped: adiciona background nas linhas ímpares

table-hover: adiciona background ao passar o mouse
