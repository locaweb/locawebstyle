Calendários

Inputs com integração de formulário.

Para o calendário utilizamos o módulo do JQuery UI.

E para aplicar basta seguir alguns passos:

1. Crie a estrutura abaixo.
2. Lembre-se de adicionar a classe datepicker.
3. Tome um suco de laranja. :D

Limitador de data

Com o limitador de data, você pode definir a data minima entre dois inputs relacionados. Para esssa funcionalidade é necessário inserir as classes .date-range englobando os inputs junto com as classes .to-date ou .from-date de acordo com os exemplos abaixo.

Data Minima

Utilize o atributo data-min-date="" para definir uma data minima.

Data Máxima

Utilize o atributo data-max-date="" para definir uma data maxima.
