Ícones

Inserção de ícones em elementos de layout.

O Bootstrap já tem um esquema de inserção de ícones em elementos. No Locaweb Style modificamos esse funcionamento para que não fosse necessário adicionar uma tag a mais no HTML.

Nós também temos a necessidade de ter várias cores para os serviços. Por isso utilizamos o serviço IcoMoon para criar um pacote dos ícones.

Atualmente, utilizamos duas fontes criadas através do IcoMoon, uma é para ícones comuns (locaicons) e a outra é apenas para ícones dos produtos internos da locaweb (panelicons).

Funcionamento

Para adicionar seu ícone, insira uma das classes da nossa coleção de ícones no elemento escolhido. Se quiser ter mais flexibilidade de posicionamento ou qualquer outra necessidade, utilize as classes em um elemento span. Mas não recomendamos muito, já que você vai precisar criar um elemento vazio só para inserir um ícone.

Vale lembrar que os ícones são inseridos no :after e :before do elemento.

Ícones a direita do texto

Por padrão os ícones ficam posicionados à esquerda. Para posicioná-los à direita, basta colocar a classe ico-pos-right.

Um exemplo: ico-home
