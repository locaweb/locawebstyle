Tabelas

Para exibir dados tabulares. Também seguem os padrões do Bootstrap, mas modificamos o seu design com uma classe nossa.

Usamos tabelas para listar informações tabulares o tempo inteiro. Os padrões do Bootstrap ainda continuam, logo, se quiser utilizar o estilo de tabelas que o Bootstrap oferece, fique à vontade. Se quiser a nossa versão, basta inserir a classe tableList na tabela.

Perceba que para deixar os títulos de colunas com marcação de ordenação, basta utilizar as classes dataDescending ou dataAscending.
