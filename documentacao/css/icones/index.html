Ícones

Coleção de font icons ativadas por classe

Funcionamento

Para adicionar seu ícone, insira uma das classes da nossa coleção de ícones no elemento escolhido. Se quiser ter mais flexibilidade de posicionamento ou qualquer outra necessidade, utilize as classes em um elemento span. Mas não recomendamos muito, já que você vai precisar criar um elemento vazio só para inserir um ícone.

Vale lembrar que os ícones são inseridos no :after e :before do elemento.

Ícones a direita do texto

Por padrão os ícones ficam posicionados à esquerda. Para posicioná-los à direita, basta colocar a classe .ls-ico-right.
