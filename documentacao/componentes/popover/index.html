No mobile, vira modal

Quando resolução abaixo de 769px o popover muda seu comportamento e é exibido como uma modal, evitando problemas de usabilidade!

Como usar

Nome Tipo Padrão Descrição
data-trigger click / hover click cria o popover baseado na posição do elemento
data-placement top / bottom / left / right top posiciona o popover
data-title text -- insere o título no popover
data-container classe ou id body posiciona o popover de acordo com elemento clicado
data-content texto e/ou tags HTML -- insere o conteúdo no popover
data-custom-classes classe -- permite inserir qualquer classe para customização do popover
