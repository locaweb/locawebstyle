Modal

Boxes simplificados e flexíveis de diálogo que necessitam do minímo de funcionalidades para funcionar.

Nome Tipo Padrão Descrição
target #myTarget -- Abre a modal com o mesmo nome do target
content um texto curto -- Abre a modal com o texto inserido

Só funciona se não houver data-target
title um titulo curto -- Abre a modal com o titulo inserido

Só funciona se não houver data-target
save texto -- Coloca um texto no botão de salvar

Só funciona se não houver data-target
close texto -- Coloca um texto no botão de fechar

Só funciona se não houver data-target
action link -- Coloca um link no botão de salvar

Só funciona se não houver data-target
class texto -- Coloca uma classe no botão de salvar

Só funciona se não houver data-target
data-custom-attrs texto -- Adiciona qualquer novo atributo no botão(type='button')

Só funciona se não houver data-target

Abrindo a modal via URL

Para abrir a modal automaticamente assim que a página for carregada, basta passar na URL o nome da modal. Tente aqui mesmo, no final da URL desta página, adicione #myAwesomeModal e aperte ENTER.
