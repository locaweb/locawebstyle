Alternando Textos

Alterando textos de elementos

Basicamente essa funcionalidade altera o texto de determiando elemento.

Classe / Atributo Descrição
data-toggle-txt Atributo cujo valor guarda o texto alternativo
data-target Elementos que terá o texto modificado pelo data-toggle-text 
1
<button class="ls-btn" data-toggle-text="Clique para voltar ao texto original">Clique para alterar o texto</button>

Se você quiser alternar o texto de outro elemento, siga o exemplo abaixo:



Uma mensagem bonita para caramba! 
1
2
3
4
<button class="ls-btn-primary" data-toggle-text="Clique para voltar ao texto original" data-target-text="#lsBox">Clique para alterar o texto do botão ao lado</button>
